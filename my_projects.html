Project Description

Over the course of July this summer I developed a propriety calibration application for A&D Precision. The application will be used by the quality division of A&D Precision to meet ISO 9001 and customer quality needs. To ensure these needs are meet, the application is able to create a database of all equipment in the factory along with every calibration done on the instruments. In order to easily find information on calibration equipment, the application is also able to display any individual bit of data or all data, showing when calibrations are due on a calendar, and generating official reports.

Project Tools

HTML, CSS, JavaScript, SQLite, Sass, Photon, Electron, NodeJs, jQuery

Project Description

At YHacks 2016, I worked with a group of three of my fellow students in order to attempt the iRobot challenge. The team at iRobot provided their iRobot Create 2 robot and Python library as tools to attempt to create a web interface allowing the robot to be manually driven while outputting sensor data and plotting its (x,y) position over time. To accomplish this challenge, we utilized a combination of Python and Flask to control the robot and communicate with the website as well as HTML, CSS, JavaScript, jQuery, and jQuery UI to create the web interface. In this challenge, we won third place.

Project Tools

iRobot Create 2, HTML, CSS, JavaScript, Python, Flask, jQuery, jQuery UI

Project Description

For the 2015-2016 Beginner Sumobot Competition, I worked with three of my fellow first year engineering students to create a 10x10cm autonomous robot with a maximum weight of 500g. The robot would have to face off against other similar robots in a sumo-style competition, where the objective was for our robot to push the other teams' robot out of the arena. To accomplish this goal, we programmed an Arduino to interface with infrared sensors and an H-Bridge Integrated Circuit through which two motors were controlled, the infrared sensors were set to be able to detect the other robot to push it while also to detect the white line around the arena to know when to turn. Our robot did well, making it into the semi-finals before being eliminated after a multiple-tie match.

Project Tools

Arduino Uno R3, Infrared Sensors, L293D, Gear Motors, Arduino Library

Project Description

As part of the University of Toronto course MIE1080 Introduction to Healthcare Reports, my partner and I wrote a review paper on the current state of robotics for use in spinal cord injury recovery and rehabilitation. As part of this review paper, we compared traditional methods, such as conventional overground walking gait therapy, with modern methods, such as using exoskeletons. For this, we chose a variety of different modern robotic systems and obtained results of traditional methods. We compared the results of these methods in order to come to a conclusion on the current state of the industry and advantages of robotic methods. The included report can be reviewed for details on the comparisons and conclusions we made.

Project Description

For the University of Toronto Institute of Aerospace Studies course, AER1515 Perception for Robotics, a fellow student and I created a clothing fold line and type classification system called Autofold. The system made use of machine learning for classification, with TensorFlow, and traditional edge detection for folding lines, with OpenCV. Both systems were written in Python and used images from the DeepFashion database for training and testing. The included report provides more detail on the systems developed and the outcomes observed.

Project Tools

Python, TensorFlow, Machine Learning, OpenCV
